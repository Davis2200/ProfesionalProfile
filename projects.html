Banking Churn Predictive System
Machine Learning Classification | Production-Ready

📊 Business Impact

Customer acquisition costs 5-25x more than retention. Lost customer lifetime value averages $10,000+. This system shifts from reactive (analyzing departing customers) to proactive (identifying at-risk accounts weeks before departure).

💰 Annual Impact: 850 retained customers × $10,000 LTV = $8.5M recovered value | 82% precision reduces wasted retention spend

🔧 Technical Approach

Pipeline: Raw customer data → Feature engineering (temporal, balance volatility, engagement) → StandardScaler normalization → XGBoost classification → Probability scores + SHAP explainability

  • XGBoost gradient boosted trees with class weighting for imbalance
  • Rolling windows capture account health trajectories
  • StandardScaler ensures fair feature comparison
  • JSON model serialization for production deployment
